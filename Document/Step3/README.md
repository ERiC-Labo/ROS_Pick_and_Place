# シミュレーション環境の解説
## シミュレーション環境起動方法
- Ctrl+Alt+Tでターミナルを開く
- cd ~/Catkin_ws/
- source devel/setup.bash
- roslaunch pick_and_place spawn_gazebo_pick_and_place.launch
- 下の図のような画面が表示されたら成功です。
